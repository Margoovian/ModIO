## Controls

- Position X: `Frequency`
- Position Y: `Volume`
- Key X: `Next Generator`
- Key Z: `Next Generator`
- Scroll: `Zoom`
- Click/Drag: `Move Position`
	

## Setup

- Include: `"ModIO/ModIO/include"`
- Link: `ModIO/bin/[*Build*]/ModIO.lib`
- Preprocessors: 
	- Build Platform: `MOD_PLATFORM_WINDOWS`
- Build:
	- `MOD_DEBUG` 
	- `MOD_RELEASE`
	- `MOD_DIST`
